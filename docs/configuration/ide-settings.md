# IDE Settings

EasyApi settings can be accessed via **Preferences(Settings)** > **Other Settings** > **EasyApi**.

## General Settings

| Setting | Description |
|---------|-------------|
| **Postman Token** | Your Postman API token for direct upload |
| **Postman Collection ID** | Target collection ID for export |
| **Postman Workspace ID** | Target workspace ID |

## Recommended Settings

EasyApi comes with built-in recommended settings that work well for most projects. These include:

- Auto-detection of Spring Boot controllers
- Resolution of common generic types
- Loading of Spring Boot configuration properties
- Support for javax.validation annotations

You can view the recommended settings by clicking **Show Recommended Config** in the settings panel.

## Method Doc

The **Method Doc** feature allows you to generate documentation for individual methods, not just API endpoints.

Enable it in **Preferences(Settings)** > **Other Settings** > **EasyApi** > **Method Doc**.

## HTTP Client

Configure the built-in HTTP client settings:

| Setting | Description |
|---------|-------------|
| **Connect Timeout** | Connection timeout in milliseconds |
| **Read Timeout** | Read timeout in milliseconds |
| **Proxy** | HTTP proxy settings |
